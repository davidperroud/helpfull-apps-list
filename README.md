## AI & Agents
- **[Sure](https://github.com/we-promise/sure)** — Application de finances personnelles avec assistant IA
- **[Baiss](https://github.com/Tbeninnovation/Baiss)** — Exécute des LLMs et RAG localement sur desktop
- **[StenoAI](https://ruzin.github.io/stenoai/)** — Transcription et summarisation de réunions localement
- **[Raven](https://github.com/31d4r/Raven)** — Chat local avec documents pour macOS/iOS
- **[KnowNote](https://github.com/MrSibe/KnowNote)** — Alternative locale à Google NotebookLM sans Docker
- **[Wifi Densepose](https://github.com/ruvnet/wifi-densepose)** — Détecte les poses humaines à travers les murs via WiFi
- **[Auto-Claude](https://github.com/AndyMik90/Auto-Claude)** — Framework multi-agent pour développement logiciel
- **[Eigent](https://github.com/eigent-ai/eigent)** — Workforce multi-agent open source local
- **[UI-TARS-desktop](https://github.com/bytedance/UI-TARS-desktop)** — Agent d'automatisation desktop local puissant (apps, navigation, fichiers) – 100% offline, multimodal
- **[5ire](https://github.com/nanbingxyz/5ire)** — Assistant IA desktop + client MCP
- **[llm-god](https://github.com/czhou578/llm-god)** — Prompt plusieurs LLMs en parallèle
- **[superpowers](https://github.com/obra/superpowers)** — Bibliothèque de compétences avancées pour Claude Code
- **[Agentfy](https://github.com/Agentfy-io/Agentfy)** — Système multi-agent pour automatisation réseaux sociaux via MCP
- **[Acontext](https://github.com/memodb-io/Acontext)** — Agents IA qui apprennent de leurs erreurs
- **[openwork](https://github.com/different-ai/openwork)** — Alternative open-source à Claude Cowork (agent macOS)
- **[Personal AI Infrastructure](https://github.com/danielmiessler/Personal_AI_Infrastructure)** — Infrastructure personnelle pour AI (Claude-powered bundles & packs)
- **[AutoMem](https://github.com/verygoodplugins/automem)** — Système de mémoire intelligent pour assistants IA
- **[RalphBlaster](https://github.com/saasmakermac/ralphblaster)** — Workflow dev full agentique : ticket → PRD → code auto (Claude-powered)
- **[PageDot CLI](https://github.com/pagedotapp/cli)** — Automatise les actions pour les emails entrants
- **[PersonaPlex-7B](https://github.com/NVIDIA/personaplex)** — Modèle conversationnel full-duplex open-source (NVIDIA)
- **[Dash](https://github.com/agno-agi/dash)** — Self-learning data agent qui s'améliore à chaque exécution
- **[Sokuji](https://github.com/kizuna-ai-lab/sokuji)** — Traduction vocale en temps réel avec IA
- **[nanobot](https://github.com/HKUDS/nanobot)** — Assistant IA personnel ultra-léger inspiré de Clawdbot
- **[DeepCode](https://github.com/HKUDS/DeepCode)** — Codage vibe avec IA : paper2code, text2web, etc.
- **[oz-workspace](https://github.com/warpdotdev/oz-workspace)** — Espace de travail collaboratif pour agents codants
- **[Moltis](https://github.com/moltis-org/moltis)** — Assistant IA personnel en Rust inspiré d'OpenClaw
- **[picoclaw](https://github.com/sipeed/picoclaw)** — Assistant IA complet sur 10MB RAM
- **[mimiclaw](https://github.com/0xSojalSec/mimiclaw)** — MimiClaw : Assistant IA sur puce ESP32-S3 à 5$
- **[open-cowork](https://github.com/OpenCoworkAI/open-cowork)** — Agent IA pour automatisation desktop et gestion de fichiers
- **[easyclaw](https://github.com/gaoyangz77/easyclaw)** — Application desktop pour agents IA
- **[antigravity-awesome-skills](https://github.com/sickn33/antigravity-awesome-skills)** — Bibliothèque de compétences pour agents IA
- **[tinyclaw](https://github.com/TinyAGI/tinyclaw)** — Équipe d'agents IA personnels qui collaborent
- **[visual-explainer](https://github.com/nicobailon/visual-explainer)** — Génère des pages HTML riches pour expliquer architecture, diffs, plans…
